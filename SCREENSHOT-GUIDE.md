## Design Guidelines

### Brand Colors
- Primary: #667eea (purple-blue)
- Secondary: #764ba2 (purple)
- Success: #10b981 (green)
- Text: #1f2937 (dark gray)
- Background: #ffffff (white) or gradient

### Typography
- Headlines: Bold, sans-serif (Inter, SF Pro Display, or similar)
- Body text: Regular, sans-serif
- Sizes: Clear hierarchy with good readability at 1280x800

### Visual Style
- Clean, modern interface
- Plenty of white space
- Subtle shadows for depth
- Consistent icon style
- Professional, not cluttered

### Screenshot Best Practices
1. Use realistic data (actual video titles from your library)
2. Show completed states (processed videos, generated reports)
3. Include success indicators (green checkmarks, progress bars)
4. Use consistent styling across all screenshots
5. Ensure text is readable at 1280x800
6. No personal information visible
7. Chrome browser chrome visible (optional but recommended)

## Tools for Creating Screenshots

### Recommended Tools
1. **Figma** (Free) - Design mockups
2. **Chrome DevTools** - Capture actual extension
3. **Cleanshot X** (Mac) - Annotate screenshots
4. **Canva** (Free tier) - Create promotional tiles
5. **Photopea** (Free web app) - Image editing

### Process
1. Use extension popup in Chrome (set to 1280x800 viewport)
2. Populate with realistic test data
3. Capture with Cleanshot X or Chrome DevTools
4. Add annotations in Figma or Canva
5. Export as PNG at exact required dimensions
6. Compress with TinyPNG to reduce file size
